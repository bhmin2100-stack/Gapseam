# 에뮬레이터 01 업데이트: PPT 이미지 중심 수정

## 요청

- PPT를 설명문 위주로 만들지 말고 실제 미니 에뮬레이터 화면과 결과 이미지를 적극적으로 사용한다.
- 가능하면 논문/공식 자료도 같이 연결해서 물리 해석의 배경을 남긴다.

## 이번에 만든 실제 에뮬레이터 이미지

- `emulator01_full_ui_direct_sputter.png`: 미니 에뮬레이터 전체 UI와 direct sputter run 결과
- `emulator01_conformal_final.png`: conformal-only 최종 profile
- `emulator01_direct_sputter_final.png`: direct sputter on 최종 profile
- `emulator01_pure_etch_final.png`: deposition 0 조건의 pure etch 결과
- `emulator01_split_strength_sweep.png`: sputter strength split sweep 결과
- `emulator01_mini_vs_gapsim_angle.png`: mini direct angle과 GapSim angle-only 비교

## PPT 반영 방향

- 1번 PPT는 같은 파일명으로 재생성했다.
- 실제 캡처 이미지를 중심 proof object로 사용했다.
- angle response는 논문 그림을 그대로 복사하지 않고 현재 미니 모델의 Gaussian response와 연결되는 설명 도식으로 재구성했다.
