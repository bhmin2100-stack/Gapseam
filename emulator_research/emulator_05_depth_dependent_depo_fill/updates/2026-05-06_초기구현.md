# Emulator 05 초기 구현

- 5번은 etch, reflected ion, sputter redeposition을 끌어오지 않는 depo-only 모델로 분리했다.
- 기본 입력 구조는 top neck가 좁고 중간이 볼록한 길쭉한 항아리형 bowing trench로 바꿨다.
- 열린 상태 deposition은 effective aspect ratio 기반 depth attenuation을 적용한다.
- closure 이후에는 sealed void를 즉시 무한 refill하지 않고 hole/line별 post-closure fill budget 안에서만 내부 void를 줄인다.
- line은 `lineOpenPathFactor`로 길이 방향 diffusion 가능성을 따로 조절한다.
