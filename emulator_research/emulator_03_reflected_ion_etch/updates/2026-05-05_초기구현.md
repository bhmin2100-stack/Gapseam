# 3번 에뮬레이터 초기 구현: one-bounce reflected ion etch

## 2026-05-05 폐기 결정

3번 reflected ion 모델은 연구 슬롯에 기록만 남기고 폐기한다.

판단 이유:

- 기본값 기준 1번 대비 최종 형상 차이가 매우 작다.
- 눈에 보일 정도로 만들려면 `Reflect %`, `Bowing`, `Microtrench`를 과하게 올려야 한다.
- 기본값도 실행 시간이 1번 대비 약 `+23%` 느려졌고, 데모용 강한 세팅은 약 `+52%` 느려졌다.
- 따라서 현재 미니 에뮬레이터에서는 3번을 채택하면 물리 해석 이득보다 속도 손실이 더 크다.

결론:

```text
3번 모델링 = 폐기
채택 모델 후보에서 제외
PPT3에는 실험 보고서와 작동 이미지, 폐기 근거만 보존
```

## 목적

3번 에뮬레이터는 1번 에뮬레이터의 `direct_angle_gaussian` sputter 결과를 보존하고, 그 위에 빠른 reduced reflected-ion etch field만 별도로 더한다.

계산 구조:

```text
depo_field
- direct_sputter_field
- reflected_ion_field
= net_growth_field
```

## 보존한 것

- 1번 direct sputter의 angle response, peak angle, width, smoothing, clamp, endpoint freeze, topology cleanup 경로를 그대로 사용했다.
- `reflected_ion_enabled = false` 또는 `reflected_ion_strength_pct = 0`이면 1번 결과와 동일하도록 테스트를 추가했다.
- 기존 GapSim `ion_los`, `redepo`, sputter preset, `deposition_pipeline`의 redeposition 구현은 3번 모델에 재사용하지 않았다.

## 새로 추가한 것

- `compute_reflected_ion_fields(...)`
- `TrenchDepoConfig.reflected_ion_*` 파라미터:
  - `reflected_ion_enabled`
  - `reflected_ion_strength_pct`
  - `reflected_ion_bowing_weight`
  - `reflected_ion_microtrench_weight`
  - `reflected_ion_range_a`

## Reduced Model

기본 경로는 full ray tracing이 아니라 zone-weighted one-bounce 근사다.

1. direct sputter field가 있는 sloped facet/sidewall을 reflection source 후보로 본다.
2. top flat은 source와 target 모두에서 강하게 억제한다.
3. target은 mid-sidewall bowing zone과 bottom-corner microtrenching zone에 집중한다.
4. source centroid와 target 사이의 거리 기반 감쇠를 곱한다.
5. reflected field는 음수가 될 수 없고, direct source가 0이면 0이 된다.

## Debug Field

`result.meta["reflected_ion_debug_fields_last"]`에 다음 field가 분리 저장된다.

- `depo_field`
- `direct_sputter_field`
- `reflection_source_field`
- `reflected_ion_field`
- `total_etch_field`
- `net_growth_field`

요약값은 `result.meta["reflected_ion_debug_summary_last"]`에 저장된다.

## 2026-05-05 UI 보정

초기 연결에서 3번 UI가 1번 direct sputter 컨트롤을 상속하지 못하고 있었다.
수정 후 3번은 다음을 동시에 표시한다.

- 1번 스퍼터 입사각 기반 에치 파라미터: `Etch A/CYC`, `Peak %`, `Peak`, `Width`, `Smooth A`
- 1번 `Sputter Gaussian` 그래프
- 3번 reflected ion 파라미터: `Reflect %`, `Bowing`, `Microtrench`, `Range A`

이 보정은 모델 의미상 중요하다.
3번은 reflected ion 단독 모델이 아니라 `1번 direct_sputter_field + reflected_ion_field`이므로,
UI와 발표자료에서도 direct sputter 기반 그래프가 먼저 보여야 한다.
